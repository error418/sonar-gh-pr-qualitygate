Sponsoring

This project tries to raise funds by utilizing the Sponsorware approach. Some features are only accessible by sponsors until specific goals are met. The feature is released to the public as soon as the goal is reached. Swingletree works perfectly fine without these features and remains accessible to everyone.

I hope you understand, that this approach enables me to maintain and further develop Swingletree more efficiently while still being able to release software to the open source community.

Sponsorware features are marked in the documentation using the term Sponsorware.

Sponsorware Goals

Reaching 4 Sponsors will release Swingletree's JUnit Plugin to the public.
