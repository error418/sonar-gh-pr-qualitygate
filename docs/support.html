Sponsorware

This project tries to raise funds by utilizing the Sponsorware approach. A public release of some features is postponed until specific sponsoring goals are met, while Sponsors get an early access to these features. Swingletree works perfectly fine without these features and remains accessible to everyone.

I hope you understand, that this approach enables me to maintain and further develop Swingletree more efficiently while still being able to release software to the open source community.

Sponsorware features are marked in the docs using the term Sponsorware.
