Core Features

GitHub Integration

No need to jump between tools to see what failed your Quality Gate. Swingletree uses the GitHub Checks API to annotate findings in your repositories.

Persisted

Build reports are persisted in GitHub using Check-Runs, which are attached to the commit. This way you are also able wo view the results of past build reports.

Ease of use

Install to your GitHub organization and you are ready to go. No machine user required for GitHub Enterprise instances.

Integrations

SonarQube Developer Edition

Swingletree can act as a SonarQube webhook endpoint. It will consume the SonarQube API to acquire additional data to assemble the build report (see showcase).

Twistlock Scan Reports

Swingletree can process Twistlock Security Scan Reports.

OWASP Zap Reports

Swingletree can process OWASP Zap Scan JSON-reports.
